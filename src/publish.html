Publicering av uppgiften

Uppstart

Jag började med att skapa en mapp som jag döpte till moment1. Jag öppnade denna i VS code och öppnade terminalen i mapp-roten. I den skrev jag npm init för att, npm är en pakethanterare som hanterar tillägg till nodeJS. Efter detta så la jag till Parcel genom att skriva npm install parcel --save-dev. Anledninge till att jag skrev --save-dev är för att det endast skall användas under utvecklingen och inte till publiceringen. Parcel sköter även minifiering av filer och gör så att man kan optimera sina bilder i sin html-kod.


I package.json lägger jag till min källfil som är index.html, jag lägger även till scripts för start och build. Jag skapar även en ny mapp som heter src (source) och i den lägger jag min källfil (index.html) samt de andra html, js, css och bildfilerna. Jag lägger även till browserlist: "> 0.5%, last 2 versions, not dead" i package.json för att göra den bakåtkompatibel bland gamla webbläsare.

Utveckling

För att få en överblick översidans utseende under utvecklingsfasen så startar jag en liveserver genom att skriva npm start i terminalen. Sedan så skriver jag själva koden i VS code inom mappen src. Jag skapar även ett github repo som jag kopplar till mitt projekt. När jag blir färdig med en del av webbplatsen så trycker jag på stage all changes sedan på commit och till sist på push för att uppdatera mitt github repo. Jag lägger även till sharp.config.json i den anger jag att kvaliteten på bilderna ska vara 60%. När jag är färdig skriver jag in nmp run build i terminalen.

Publicering

Webbplatsen publiceras via netlify, där väljer jag vilket github repo som ska publiceras och ställer in de inställningar som jag vill ha. Därmed är min webbplats publicerad.
